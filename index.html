Get insights into your GitHub activity

Git Stats Board is a web application that displays GitHub user statistics in an easy-to-read format. With Git Stats Board, you can track your progress, identify trends, and stay motivated.

View on GitHub

Key Features

Language Breakdown

See which programming languages you use the most in your repositories.

Commit History

Track your commit history over time and see how your activity changes.

Repository Overview

Get an overview of your repositories, including stars, forks, and issues.

About Git Stats Board

Git Stats Board was inspired by the need for a simple and intuitive way to track GitHub activity. The project was developed over a period of 3 weeks as a Portfolio Project for Holberton School.
